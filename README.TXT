Project: Prison Break
Author: Mehmet Sofi

This project is coursework number 2 for Programming Practice and Applications
   
This project is a simple framework for an adventure game. You can move between rooms, pick and drop items.

to access certain rooms, some items need to be collected

there exists a teleporter room in which you will be teleported to a random room within the prisoner area

To win this game, player must make it to the room called outside.

To start this application, create an instance of class "Game" and call its
"play" method. or call the main method of the class GameMain.

This project is an improved version of game world of zuulu project named as zuulu-better by orginal 
authors Michael Kolling and David J Barnes
